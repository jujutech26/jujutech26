**तपाईंको मोबाइल वा laptop लाई इन्टरनेटले कसरी चिन्छ?**

जसरी घरको ठेगाना हुन्छ, त्यसरी नै इन्टरनेटमा हरेक device को पनि आफ्नो एउटा unique ठेगाना हुन्छ।

त्यो ठेगानालाई **IP address** भनिन्छ।

192.168.1.1 जस्तो देखिने नम्बरलाई नै IP Address भनिन्छ।

यस्तो IP Address ले Internet मा तपाईंको Device को पहिचान गराउँछ।

उदाहरणका लागि:
1. जब तपाईंko laptople Google मा “Balen Shah photo” search गर्नुहुन्छ,
2. तपाईंको laptople Google server लाई “मलाई Balen Shah को photo देऊ” भनेर request पठाउँछ।
3. Google server ले database मा search गरेर Balen Shah का photos फेला पार्छ।
4. अनि त्यो data तपाईंको IP address मा response को रूपमा फर्काउँछ।
5. र अन्तिममा तपाईंले आफ्नो Laptop मा त्यो Photo हेर्न सक्नुहुन्छ।

अब प्रश्न आउँछ, IP किन आवश्यक छ?
- जसरी हुलाकबाट कसैलाई चिठी पठाउन ठेगाना बिना सम्भव हुँदैन,
- त्यसरी नै IP Address bina kasaile इन्टरनेटमा request गर्न र response पाउन सम्भव हुँदैन।
